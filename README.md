チームm_cre (プロトコル部門)
====

[人狼知能プレ大会@GAT2017](http://aiwolf.org/event-2/gat2017) プロトコル部門の出場エージェントです。

## 必要ライブラリ等

### ライブラリ
* aiwolf-client.jar (0.4.4)
* aiwolf-common.jar (0.4.4)
* aiwolf-server.jar (0.4.4)
* aiwolf-viewer.jar (0.4.4)
* jsonic-1.3.10.jar
